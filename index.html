Web Development Flashcards

HTML

<html>

All HTML code goes inside a <html> and </html>

<head>

The code inside a <head> and </head> doesn't show up on the website, tags like <title> do show up in the tab at the top of the browser.

<body>

All the code and content on a website goes inside a <body> and </body>

<h1>

This is the title, or main headline for a website. Use <h1> and an </h1>. Each page can only have one.

<h2>

This is the subtitle, or sub-headline for a website. Use <h2> and an </h2>

<h3>

This is the subtitle's subtitle for a website. Use <h3> and an </h3>

<p>

Use <p> and an </p> around each paragraph you write out.

<ul>

Use <ul> and an </ul> a bulleted list. Each list item is always inside of a unordered list tags: <li> </li>

<li>

Use <li> and an </li> for items on a list. A list item is always inside of a unordered list <ul> </ul>

<div>

Use <div> and </div> with text and photos inside of it for each section of a website.

<img>

This is an image tag. It doesn't have a closing tag, so it looks like this and should include the location of a photo: Use <img src="photo.jpg">
